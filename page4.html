CSS POSITIONS

The Position property

The position property specifies the type of positioning method used for an element.

There are five different position values:
1. static
2. relative
3. fixed
4. absolute
5. sticky

Elements are then positioned using top, bottom ,left and right
properties.However these properties will not work unless the
postion property is set first.They also work differently
depending on the position value .
